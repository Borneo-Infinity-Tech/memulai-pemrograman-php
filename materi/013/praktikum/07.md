# Soal Praktikum 7: Menentukan Tarif Listrik Rumah Tangga

### **Studi Kasus**

Sebuah program PHP berbasis terminal digunakan untuk menentukan tarif listrik berdasarkan daya listrik rumah dan jumlah pemakaian kWh.

**Aturan Tarif**

- Jika **daya listrik = 900 VA**:
  - Jika **pemakaian ≤ 100 kWh**, tarif **Rp1.000 per kWh**
  - Jika **pemakaian > 100 kWh**, tarif **Rp1.200 per kWh**
- Jika **daya listrik = 1300 VA**:
  - Jika **pemakaian ≤ 100 kWh**, tarif **Rp1.400 per kWh**
  - Jika **pemakaian > 100 kWh**, tarif **Rp1.600 per kWh**

---

### **Algoritma Narasi**

1.  Mulai program.
2.  Minta pengguna memasukkan **daya listrik rumah (900 atau 1300 VA)** melalui terminal menggunakan fungsi `readline()`.
3.  Simpan nilai daya listrik ke dalam variabel `daya`.
4.  Minta pengguna memasukkan **jumlah pemakaian listrik (dalam kWh)** melalui terminal menggunakan fungsi `readline()`.
5.  Simpan jumlah pemakaian ke dalam variabel `pemakaian`.
6.  Periksa apakah `daya` bernilai **900**.
    - Jika **ya**, lanjutkan ke langkah 7.
    - Jika **tidak**, lanjutkan ke langkah 8.
7.  Periksa jumlah pemakaian untuk daya 900 VA.
    - Jika `pemakaian` kurang dari atau sama dengan 100 kWh, tampilkan **"Tarif Rp1.000 per kWh"**.
    - Jika lebih dari 100 kWh, tampilkan **"Tarif Rp1.200 per kWh"**.
8.  Periksa jumlah pemakaian untuk daya 1300 VA.
    - Jika `pemakaian` kurang dari atau sama dengan 100 kWh, tampilkan **"Tarif Rp1.400 per kWh"**.
    - Jika lebih dari 100 kWh, tampilkan **"Tarif Rp1.600 per kWh"**.
9.  Selesai.

### **Instruksi**

1.  **Salin kode PHP di bawah ini**
2.  Simpan dengan nama `percabangan_bersarang_07.php`
3.  Jalankan pada terminal
4.  Masukkan nilai sesuai permintaan program
5.  Perhatikan hasil output
