# Soal Praktikum 1: Percabangan Bersarang Sederhana

### **Studi Kasus**

Sebuah program digunakan untuk menentukan **status kelulusan siswa** berdasarkan:

1.  **Nilai ujian**
2.  **Kehadiran siswa**

Ketentuan:

- Jika **nilai ≥ 75**
  - Jika **kehadiran ≥ 80 maka LULUS**
  - Jika **kehadiran \< 80 maka TIDAK LULUS (Kurang Kehadiran)**
- Jika **nilai \< 75 maka TIDAK LULUS (Nilai Kurang)**

### **Instruksi**

1.  **Salin kode PHP di bawah ini**
2.  Simpan dengan nama `percabangan_bersarang_01.php`
3.  Jalankan pada terminal
4.  Perhatikan hasil output

```php
<?php
$nilai = 80;
$kehadiran = 85;

if ($nilai >= 75) {
    if ($kehadiran >= 80) {
        echo "Status: LULUS";
    } else {
        echo "Status: TIDAK LULUS (Kehadiran kurang)";
    }
} else {
    echo "Status: TIDAK LULUS (Nilai kurang)";
}
?>
```

**Output yang di harapkan**

```php
Status: LULUS
```

**Tugas Tambahan**

1.  Ganti nilai pada variabel nilai dan kehadiran dengan angka lain.
2.  Jalankan ulang kode, perhatikan hasil output (coba sampai semua jenis output tampil seperti `Status: TIDAK LULUS (Kehadiran kurang)` dan `Status: TIDAK LULUS (Kehadiran kurang)`
3.  Pahami alur kode yang ditulis
