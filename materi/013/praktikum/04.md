# Soal Praktikum 4: Menentukan Hak Akses Pengguna

### **Studi Kasus**

Sebuah sistem login menentukan **hak akses pengguna** berdasarkan **peran (role)** dan **status akun**.

**Aturan Hak Akses**

- Jika **role = admin**:
  - Jika **status = aktif**, tampilkan **"Akses penuh"**
  - Jika **status = tidak aktif**, tampilkan **"Akun diblokir"**
- Jika **role = user**:
  - Jika **status = aktif**, tampilkan **"Akses terbatas"**
  - Jika **status = tidak aktif**, tampilkan **"Akun diblokir"**

---

### **Algoritma Narasi**

1.  Mulai program.
2.  Simpan peran pengguna (`admin` atau `user`) ke dalam variabel `role`.
3.  Simpan status akun (`aktif` atau `tidak aktif`) ke dalam variabel `statusAkun`.
4.  Periksa apakah `role` adalah **admin**.
    - Jika **ya**, lanjutkan ke langkah 5.
    - Jika **tidak**, lanjutkan ke langkah 6.
5.  Periksa status akun admin.
    - Jika `statusAkun` adalah **aktif**, tampilkan **"Akses penuh"**.
    - Jika **tidak aktif**, tampilkan **"Akun diblokir"**.
6.  Periksa status akun user.
    - Jika `statusAkun` adalah **aktif**, tampilkan **"Akses terbatas"**.
    - Jika **tidak aktif**, tampilkan **"Akun diblokir"**.
7.  Selesai.

### **Instruksi**

1.  **Salin kode PHP di bawah ini**
2.  Simpan dengan nama \`percabangan_bersarang_04.php\`
3.  Jalankan pada terminal
4.  Perhatikan hasil output
