# Soal Praktikum 2: Menentukan Jenis Diskon Pembelian

### **Studi Kasus**

Sebuah toko memberikan **diskon berdasarkan total belanja** dan **status member**.

### **Aturan Diskon**

- Jika **total belanja â‰¥ 500.000**:
  - Jika **member = ya**, diskon **20%**
  - Jika **member = tidak**, diskon **10%**
- Jika **total belanja \< 500.000**, **tidak mendapat diskon**

---

### **Algoritma Narasi**

1.  Mulai program
2.  Simpan total belanja ke dalam variabel `totalBelanja`.
3.  Simpan status member (`ya` atau `tidak`) ke dalam variabel `statusMember`.
4.  Periksa apakah `totalBelanja` lebih besar atau sama dengan 500.000.
    - Jika **ya**, lanjutkan ke langkah 5.
    - Jika **tidak**, tampilkan **"Tidak mendapat diskon"**, lalu program selesai.
5.  Periksa apakah `statusMember` bernilai **"ya"**
    - Jika **ya**, tampilkan **"Diskon 20%"**.
    - Jika **tidak**, tampilkan **"Diskon 10%"**.
6.  Selesai

### **Instruksi**

1.  **Salin kode PHP di bawah ini**
2.  Simpan dengan nama `percabangan_bersarang_02.php`
3.  Jalankan pada terminal
4.  Perhatikan hasil output
