# Soal Praktikum 8: Hitung Nilai Akhir dengan Percabangan Bersarang

### **Studi Kasus**

Sistem penilaian siswa menggunakan:

- **Nilai Tugas** (40%)
- **Nilai UTS** (30%)
- **Nilai UAS** (30%)

Rumus nilai akhir:

```
nilai_akhir = (tugas × 0.4) + (uts × 0.3) + (uas × 0.3)
```

Ketentuan kelulusan:

- Jika **nilai akhir ≥ 75**
  - Jika **nilai UAS ≥ 60** → **LULUS**
  - Jika **nilai UAS \< 60** → **TIDAK LULUS (UAS kurang)**
- Jika **nilai akhir \< 75** → **TIDAK LULUS (Nilai akhir kurang)**

---

### **Instruksi**

1.  Salin kode berikut
2.  Simpan dengan nama `nilai_akhir_bersarang.php`
3.  Jalankan di terminal
4.  Masukkan nilai sesuai perintah

---

### **Kode Program**

```php
<?php
$tugas = readline("Masukkan nilai tugas: ");
$uts   = readline("Masukkan nilai UTS: ");
$uas   = readline("Masukkan nilai UAS: ");

$tugas = (float)$tugas;
$uts   = (float)$uts;
$uas   = (float)$uas;

$nilai_akhir = ($tugas * 0.4) + ($uts * 0.3) + ($uas * 0.3);

echo "Nilai Akhir: $nilai_akhir \n";

if ($nilai_akhir >= 75) {
    if ($uas >= 60) {
        echo "Status: LULUS \n";
    } else {
        echo "Status: TIDAK LULUS (Nilai UAS kurang) \n";
    }
} else {
    echo "Status: TIDAK LULUS (Nilai akhir kurang) \n";
}
?>
```

### **Contoh Input dan Output**

```
Masukkan nilai tugas: 80
Masukkan nilai UTS: 75
Masukkan nilai UAS: 70

Nilai Akhir: 75.5
Status: LULUS
```
