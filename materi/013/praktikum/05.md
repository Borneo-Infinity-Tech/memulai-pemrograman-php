# Soal Praktikum 5: Percabangan Bersarang dengan Input Terminal

### **Studi Kasus**

Sebuah program digunakan untuk menentukan **status kelulusan siswa** berdasarkan:

1.  Nilai ujian
2.  Kehadiran siswa

Ketentuan:

- Jika **nilai ≥ 75**
  - Jika **kehadiran ≥ 80** → **LULUS**
  - Jika **kehadiran \< 80** → **TIDAK LULUS (Kehadiran kurang)**
- Jika **nilai \< 75** → **TIDAK LULUS (Nilai kurang)**

### **Instruksi**

1.  **Salin kode PHP di bawah ini**
2.  Simpan dengan nama `percabangan_bersarang_05.php`
3.  Jalankan pada terminal
4.  Masukkan nilai sesuai permintaan program
5.  Perhatikan hasil output

### **Kode Program**

```php
<?php
$nilai = readline("Masukkan nilai ujian: ");
$kehadiran = readline("Masukkan persentase kehadiran: ");

$nilai = (int)$nilai;
$kehadiran = (int)$kehadiran;

if ($nilai >= 75) {
    if ($kehadiran >= 80) {
        echo "Status: LULUS \n";
    } else {
        echo "Status: TIDAK LULUS (Kehadiran kurang) \n";
    }
} else {
    echo "Status: TIDAK LULUS (Nilai kurang) \n";
}
?>
```

---

**Contoh Input dan Output**

```php
Masukkan nilai ujian: 80
Masukkan persentase kehadiran: 85

Status: LULUS
```
