# Soal Praktikum 1: Perulangan `while`

Mengenal struktur dasar perulangan `while` dalam PHP.

**Instruksi**

1.  Buat file PHP dengan nama `while1.php`
2.  Salin kode PHP di bawah ini dan Jalankan file tersebut
3.  Amati hasil yang ditampilkan

```php
<?php
$no = 1;

while ($i <= 5;) {
    echo "Angka ke-$i \n";

    $i++;
}
?>
```

Output yang diharapkan

```
Angka ke-1
Angka ke-2
Angka ke-3
Angka ke-4
Angka ke-5
```

Silahkan ganti angka 5 pada perulangan sesuai angka yang kalian inginkan, jalankan ulang program dan perhatikan outputnya.
