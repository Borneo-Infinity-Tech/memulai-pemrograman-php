# Latihan Pengenalan Array

**Kerjakan soal di bawah ini dengan jawaban yang tepat!**

## Soal 1: Mengubah Data ke Array

Diberikan data berikut:

- Andi
- Budi
- Citra

**Tugas:**

1.  Simpan data tersebut ke dalam array indeks di PHP.
2.  Tampilkan nama **Budi** dari array tersebut.

---

## Soal 2: Membuat dan Mengakses Array

Buatlah array yang berisi angka:  
`10, 20, 30, 40, 50`

**Tugas:**

1.  Simpan ke dalam array.
2.  Tampilkan angka pertama.
3.  Tampilkan angka terakhir.

---

## Soal 3: Menampilkan Isi Array

Diberikan array:

```php
$hewan = ["Kucing", "Anjing", "Kelinci"];
```

**Tugas:**

1.  Tampilkan semua isi array menggunakan `echo`.

---

## Soal 4: Menambah Data ke Array

Buat array yang berisi:  
`Merah, Biru`

**Tugas:**

1.  Tambahkan warna **Hijau** ke dalam array.
2.  Tampilkan semua isi array.

---

## Soal 5: Studi Kasus (Data Nyata ke Array)

Diberikan data nilai:  
`75, 80, 85`

**Tugas:**

1.  Simpan data tersebut dalam array.
2.  Tampilkan nilai ke-2.
3.  Hitung jumlah semua nilai.

---

## Soal 6: Manipulasi Indeks

Diberikan array:

```php
$angka = [2, 4, 6, 8, 10];
```

**Tugas:**

1.  Tampilkan angka ke-4.
2.  Ubah nilai angka ke-2 menjadi 20.
3.  Tampilkan seluruh isi array setelah diubah.

---

## Soal 7: Menghitung Rata-rata

Buat array berisi nilai:  
`80, 85, 90, 75, 70`

**Tugas:**

1.  Hitung jumlah seluruh nilai.
2.  Hitung rata-rata nilai.
3.  Tampilkan hasilnya.

---

## Soal 8: Mengurutkan dan Menampilkan

Diberikan array:

```php
$angka = [12, 5, 20, 8, 15];
```

**Tugas:**

1.  Urutkan array dari yang terkecil ke terbesar.
2.  Tampilkan hasilnya.
3.  Tampilkan juga nilai terbesar dan nilai terkecil.
