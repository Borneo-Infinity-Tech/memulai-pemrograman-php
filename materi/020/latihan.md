# Latihan Pengenalan Array

**Kerjakan soal di bawah ini dengan jawaban yang tepat!**

## Soal 1:

Buatlah sebuah array yang berisi 10 nama teman kamu. Gunakan perulangan `for` untuk menampilkan semua isi array tersebut satu per satu ke layar.

## Soal 2:

Diberikan sebuah array berisi angka: `[2, 4, 6, 8, 10]`. Gunakan perulangan `for` untuk menampilkan semua isi array tersebut ke layar.

## Soal 3:

Buatlah sebuah array yang berisi nama bulan dari Januari sampai Desember.

- Tampilkan semua isi array menggunakan perulangan `for`
- Gunakan perulangan `for` untuk menampilkan data dengan format nomor urut di depannya.

```php
1. Januari
2. Februari
3. Maret
....
12. Desember
```

- Tampilkan semua isi array di mulai dari Desember sampai Januari menggunakan perulangan `for`
- Tampilkan hanya bulan genap saja menggunakan perulangan `for`

## Soal 4:

Diberikan sebuah array berisi angka: `[2, 4, 6, 8, 10]`. Gunakan perulangan `for` untuk menghitung jumlah seluruh angka dalam array tersebut, lalu tampilkan hasilnya.

## Soal 5:

Buatlah dua buah array dengan panjang yang sama, misalnya:

- Array A: `[1, 2, 3, 4, 5]`
- Array B: `[5, 4, 3, 2, 1]`

Gunakan perulangan `for` untuk menampilkan hasil penjumlahan dari masing-masing index kedua array tersebut.
