# Soal Praktikum 11: FizzBuzz dari 1 sampai 50

Melatih penggunaan **perulangan** `for` dan **percabangan** `if` dengan beberapa kondisi.

**Instruksi:**

1.  Buat file PHP bernama `fizzbuzz.php`
2.  Gunakan perulangan `for` untuk menampilkan angka dari **1 sampai 50**
3.  Dengan ketentuan:
    - Jika angka habis dibagi **3**, tampilkan **"Fizz"**
    - Jika angka habis dibagi **5**, tampilkan **"Buzz"**
    - Jika angka habis dibagi **3 dan 5**, tampilkan **"FizzBuzz"**
    - Jika tidak memenuhi kondisi di atas, tampilkan angkanya
4.  Setiap hasil ditampilkan pada baris baru

**Contoh Output (sebagian):**

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
...
14
FizzBuzz
...
49
Buzz
```
