# Soal Praktikum 3: Menampilkan Teks Berulang

Memahami perulangan untuk menampilkan teks.

**Instruksi:**

1.  Buat file PHP bernama `for3.php`
2.  Gunakan perulangan `for` untuk menampilkan teks  
    **"Belajar PHP itu menyenangkan"** sebanyak **5 kali**
3.  Setiap teks ditampilkan pada baris baru

**Output yang diharapkan:**

```
Belajar PHP itu menyenangkan
Belajar PHP itu menyenangkan
Belajar PHP itu menyenangkan
Belajar PHP itu menyenangkan
Belajar PHP itu menyenangkan
```
