# Soal Praktikum 14: Menampilkan Bintang Persegi Panjang

Mengenal penggunaan **perulangan** `for` **bersarang** dalam PHP.

**Instruksi:**

1.  Buat file PHP bernama `bintang_persegi.php`
2.  **Salin dan jalankan kode berikut** tanpa mengubah apa pun
3.  Amati hasil yang ditampilkan di terminal

**Kode yang disalin:**

```php
<?php
for ($i = 1; $i <= 4; $i++) {
    for ($j = 1; $j <= 6; $j++) {
        echo "* ";
    }
    echo "\n";
}
?>
```

**Output yang diharapkan:**

```
* * * * * *
* * * * * *
* * * * * *
* * * * * *
```
