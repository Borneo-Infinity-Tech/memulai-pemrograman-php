# Soal Praktikum 13: Menampilkan Proses Penjumlahan dengan Batas Akhir Input

Melatih penggunaan **form input**, **perulangan** `for`, **penggabungan string**, dan **operasi penjumlahan**.

**Instruksi:**

1.  Buat file PHP bernama `penjumlahan_input.php`
2.  Buat sebuah **input** untuk memasukkan **angka batas akhir** menggunakan `readline()`
3.  Program akan menampilkan **proses penjumlahan dari 1 sampai angka tersebut**
4.  Gunakan perulangan `for` untuk membentuk deret penjumlahan
5.  Tampilkan hasil total penjumlahannya

**Ketentuan:**

- Format tampilan:  
  `1 + 2 + 3 + ... + n = hasil`
- Tanda `+` **tidak boleh muncul di akhir deret**
- Gunakan perulangan `for`
- Gunakan variabel untuk menyimpan total

**Output yang diharapkan:**

```
Masukkan angka batas akhir: 5

1 + 2 + 3 + 4 + 5 = 15
```
