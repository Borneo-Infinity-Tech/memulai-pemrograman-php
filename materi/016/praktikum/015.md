# Soal Praktikum 15: Menampilkan Pola Segitiga Bintang

Mengenal penggunaan **perulangan** `**for**` **bersarang** untuk membentuk pola segitiga.

**Instruksi:**

1.  Buat file PHP bernama `segitiga_bintang.php`
2.  **Salin dan jalankan kode berikut** tanpa mengubahnya
3.  Amati hasil yang ditampilkan

**Kode yang disalin:**

```php
<?php
for ($i = 1; $i <= 5; $i++) {
    for ($j = 1; $j <= $i; $j++) {
        echo "* ";
    }
    echo "\n";
}
?>
```

**Output yang diharapkan:**

```
*
* *
* * *
* * * *
* * * * *
```
