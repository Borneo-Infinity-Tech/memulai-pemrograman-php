# Soal Latihan Perulangan `for`

### **Soal 1**

Buatlah program PHP menggunakan perulangan `for` untuk menampilkan angka **1 sampai 10** ke layar.

---

### **Soal 2**

Buatlah program PHP yang menampilkan tulisan **"Belajar PHP di SMK"** sebanyak **5 kali** menggunakan perulangan `for`.

---

### **Soal 3**

Buatlah program PHP menggunakan perulangan `for` untuk menampilkan angka **10 sampai 1** (urutan menurun).

---

### **Soal 4**

Buatlah program PHP menggunakan perulangan `for` untuk menampilkan **angka genap dari 2 sampai 20**.

---

### **Soal 5**

Diketahui sebuah program PHP sebagai berikut:

```php
for ($i = 1; $i <= 5; $i++) {
    echo "PHP<br>";
}
```

- Berapa kali teks **"PHP"** akan ditampilkan?
- Jelaskan secara singkat mengapa perulangan tersebut berhenti.
